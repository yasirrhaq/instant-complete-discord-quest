# instant-complete-discord-quest

## Setup Instructions

1. **Close Discord completely**
   - Make sure Discord is fully closed (end the task from Task Manager if necessary).

2. **Enable Discord Developer Mode**
   - Press **Windows + R**
   - Type the following and press Enter:
     ```
     %appdata%/discord/
     ```
   - Open the file **`settings.json`** using Notepad.
   - Replace **all existing content** in the file with the provided [settings.json](settings.json)
   - Save the file and close Notepad.

3. **Open Discord and Access Developer Tools**
   - Launch Discord.
   - Press **CTRL + Shift + I** to open Developer Tools.

4. **Run the Script**
   - Click on the **Console** tab (top-right area).
   - Scroll down if needed.
   - Paste the provided [code](Skip%20Quest.js) into the console.
   - Press **Enter** to execute.

---

⚠️ **Note:**  
Make sure Discord is fully closed before editing `settings.json`, otherwise the changes may not apply.
Quest completion may take some time, so please be patient.
